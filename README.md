# singk_qt
LAN multi-person synchronized youtube video desktop player. using qt5.  video from youtube

<h1>library:</h1>
qt5
pytube

# build
1. use qt5 to build
2. use py2exe to pack scripts
3. put dist/* into qt5output/pythonBin/
4. done
